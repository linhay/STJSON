  - **JSON Date 默认序列化纪元隔离**：定位并记录了 Swift JSONEncoder 默认的 `.deferredToDate` 编码策略（基于 2001 年纪元，造成数值转换为 621692800），通过直接初始化 `Date(timeIntervalSinceReferenceDate: 1600000000)` 隔离了编码策略的差异。
- 评测结论：
  - 全量 211 个单元测试（新增后）全部 100% 成功通过，彻底消除了 AnyCodable 之前未被覆盖的深层类型测试盲区。

## 引入 yyjson 风格深度性能优化记录
- 关键决策：
  - **一维扁平集合惰性 Bypass 优化**：在 [SwiftyJSON.swift](file:///workspace/open-sources/STJSON/Source/SwiftyJSON/SwiftyJSON.swift#L260-L315) 中，优化了高频核心 `unwrap` 递归机制。对 `[Any]` 和 `[String: Any]` 等复杂集合加入了 O(N) 线性类型守卫扫描，当且仅当子节点中真的存在 `JSON` 实例或嵌套子集合时，才进行实际的 map 拆包与内存拷贝；对所有一维扁平原生集合（如从 JSON 序列化产生的叶子坐标数组和键值对）直接返回其本身，免去了数以万计的拷贝与 map 开销。

- 评测结论：
  - 全量 216 个测试用例已全部跑通，功能 100% 严谨安全。
