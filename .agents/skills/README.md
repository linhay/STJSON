# STJSON Development Skills

这个目录用于存放 STJSON 仓库内部维护流程的 skills。

这些 skills 只服务于仓库维护者，不进入 public release bundle。对外 skills 统一放在 `skills/`。

当前内部 skills：

- `stjson-swift-library-engineering`：STJSON / AnyCodable / SwiftyJSON 代码与测试改动流程
- `stjson-ops-governance`：文档、memory、AGENTS、skill 沉淀流程
- `stjson-release-governance`：public skills 打包与 release 契约维护

对外 public skills 统一放在 `skills/`，当前包括：

- `stjson-usage`
- `stjson-dev-feedback`
- `stjson-update`
